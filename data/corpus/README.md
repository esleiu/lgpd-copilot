# Corpus

Corpus usado no projeto: recorte estruturado da LGPD, com artigos essenciais e notas orientativas baseadas em temas comuns de compliance.

Fontes oficiais para validacao:

- Lei Geral de Protecao de Dados Pessoais, Lei 13.709/2018: https://www.planalto.gov.br/ccivil_03/_ato2015-2018/2018/lei/l13709.htm
- ANPD, guias e orientacoes: https://www.gov.br/anpd/pt-br/documentos-e-publicacoes

O corpus foi escrito em Markdown para facilitar chunking, citacao por artigo e deploy em Streamlit Cloud.
